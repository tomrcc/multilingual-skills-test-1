Blogging on CloudCannon supports **Snippets**: rich interactive components beyond standard markdown. These MDX components enable sophisticated layouts, embedded functionality, and enhanced user engagement without requiring custom development for each post.

This post demonstrates a range of sample Snippets, including placeholder text for each one. You can also <a href="https://cloudcannon.com/documentation/developer-articles/snippets-using-mdx-components/" target="_blank" rel="noopener">make your own Snippets using MDX components</a>.
