Bloguear en CloudCannon admite **Snippets**: componentes interactivos enriquecidos que van más allá del markdown estándar. Estos componentes MDX permiten diseños sofisticados, funcionalidad integrada y una mayor interacción de los usuarios sin necesidad de desarrollar código personalizado para cada publicación.

Esta publicación muestra una variedad de Snippets de ejemplo, incluido texto de marcador de posición para cada uno. También puedes <a href="https://cloudcannon.com/documentation/developer-articles/snippets-using-mdx-components/" target="_blank" rel="noopener">crear tus propios Snippets con componentes MDX</a>.
