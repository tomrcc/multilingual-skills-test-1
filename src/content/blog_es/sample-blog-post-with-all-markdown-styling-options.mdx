Puedes agregar más; consulta [https://cloudcannon.com/documentation/developer-articles/what-are-rich-text-editors/#the-wysiwyg-toolbar](https://cloudcannon.com/documentation/developer-articles/what-are-rich-text-editors/#the-wysiwyg-toolbar)

## Encabezado de nivel dos: para secciones principales

Este es el nivel de encabezado principal para las secciones más importantes dentro de las publicaciones del blog. Ofrece una jerarquía visual clara y ayuda a los lectores a escanear la estructura del contenido.

### Encabezado de nivel tres: para subsecciones

Usa este nivel para dividir las secciones principales en temas más pequeños y específicos. Mantiene la jerarquía a la vez que conserva el contenido organizado.

#### Encabezado de nivel cuatro: para desgloses detallados

Este nivel de encabezado funciona bien para ejemplos específicos, casos de estudio o explicaciones técnicas detalladas dentro de las subsecciones.

##### Encabezado de nivel cinco: rara vez necesario

La mayoría de las publicaciones de blog no requerirán este nivel de profundidad, pero está disponible para documentación técnica extremadamente detallada o guías exhaustivas.

###### Encabezado de nivel seis: el encabezado más pequeño

Este es el nivel de encabezado más profundo disponible. Úsalo con moderación y solo cuando sea realmente necesario para la organización del contenido.

## Opciones de formato de texto

**El texto en negrita** se utiliza para enfatizar, resaltar términos importantes o conceptos clave que merecen atención. Úsalo con moderación: el uso excesivo reduce su impacto.

_El texto en cursiva_ aporta un énfasis sutil, a menudo para términos técnicos, palabras extranjeras o cuando quieres destacar algo sin el peso de la negrita.

Puedes combinar formatos: **negrita y** **_cursiva_** **juntas** o **_negrita y cursiva a la vez_**. Estas combinaciones deben usarse en contadas ocasiones y solo cuando sean realmente necesarias para mayor claridad.

## Listas para organizar la información

### Listas numeradas

Usa listas numeradas cuando el orden importa o cuando presentas pasos secuenciales:

1. Primer elemento de la secuencia
2. El segundo elemento sigue de forma lógica
3. El tercer elemento continúa la progresión
4. El cuarto elemento mantiene el flujo
5. El quinto elemento completa la secuencia

Las listas numeradas funcionan bien para:

- Instrucciones paso a paso
- Comparaciones jerarquizadas
- Eventos cronológicos
- Elementos ordenados por prioridad

### Listas con viñetas

Usa listas con viñetas cuando el orden no importa:

- Primer punto de la colección
- Segundo punto de igual importancia
- Tercer punto, también sin orden
- Cuarto punto del conjunto
- Quinto punto que completa el grupo

Las listas con viñetas funcionan bien para:

- Listas de funciones
- Colecciones sin orden
- Opciones sin jerarquía
- Puntos de información general

### Listas anidadas

Puedes anidar listas dentro de listas para crear estructuras más complejas:

1. Punto principal que requiere elaboración
   - Detalle de apoyo para el punto principal
   - Información de apoyo adicional
   - Tercer detalle de apoyo
2. Segundo punto principal con subelementos
   - Detalle anidado bajo el segundo punto
   - Otro elemento anidado
3. Tercer punto principal
   - Detalle anidado final
   - Elemento anidado de cierre

## Citas para enfatizar

> Las citas son excelentes para destacar afirmaciones importantes, testimonios de clientes o conclusiones clave que merecen especial atención. Ofrecen una pausa visual respecto al texto habitual y dirigen la mirada del lector hacia la información crítica.

También puedes usar citas para citaciones extensas:

> "Nuestra infraestructura funcionó a la perfección durante el evento de mayor tráfico que jamás hayamos experimentado. El equipo de Jetstream se anticipó a nuestras necesidades y preparó la red incluso antes de que se lo pidiéramos. Ese es el tipo de ingeniería proactiva que los diferencia de cualquier otro proveedor de CDN que hayamos evaluado."
>
> — Elena Rodriguez, CTO de StreamVault Media

## Imágenes en las publicaciones del blog

Las imágenes deben usarse de forma estratégica para ilustrar conceptos, mostrar visualizaciones de datos o proporcionar pausas visuales en contenido con mucho texto.

![Figura 1: Distribución de nodos de borde en América del Norte a través de la infraestructura de red de Jetstream](/src/assets/images/blog-country.png)

_Figura 1: Distribución de nodos de borde en América del Norte a través de la infraestructura de red de Jetstream_

Al incluir imágenes, proporciona siempre:

- Texto alternativo descriptivo para la accesibilidad
- Leyendas claras que expliquen lo que muestra la imagen
- Contexto relevante que conecte la imagen con el texto circundante
